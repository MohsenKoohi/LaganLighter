#  Transform (symmetrize/transpose/randomzie) and Store in CompBin Format

This code loads the graph, and based on the user request applies transformation(s).
The output graph is stored in CompBin format.

WG2Compbin: https://github.com/MohsenKoohi/WG2CompBin

### Source code
The source code is available on [alg11_compbin.c](../alg11_compbin.c).

User can pass `-r`, `-t`, or `-s` to request randomization, transposition, and symmetrization of the input graph.
The `-o path/to/output` is also needed as an argument.
Using the `launcher.sh`, the arguments should be passed using `program-args`, for example `program-args="-r -t -o ~/data"`.
